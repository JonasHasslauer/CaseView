@author: Jonas Haßlauer

Abstract

Die Anwendung 'CaseView' soll die aktuellen Corona-Zahlen angeben. Dabei kann der User auswählen, für welchen Landkreis
diese angezeigt werden sollen. Falls möglich, sollen dazu noch die aktuellen Regeln - die mit diesem Inzidenzwert
einhergehen - mit ausgegeben werden.
